# golang_full

## Goal
Demonstrate knowledge with Go

## Requirements
- Docker 24.0.6 or a compatible version
- Docker Compose 2.23.0 or a compatible version

## How to Start
1. in your terminal command execute:
```
make init
```

2. after execute:
```
make rest
```
